# 拼多多百亿补贴团购小程序

拼多多百亿补贴里面有些拼团需要多个人拼单才能购买成功，所以打算做一个免费开源的小程序，用来帮助大家拼团购买。

目前还未上线，还没有完成写完，然后域名也没有备案。
